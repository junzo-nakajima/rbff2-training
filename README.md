# リアルバウト餓狼伝説2 トレーニングモードLuaスクリプト

1998年に発売されたネオジオの対戦格闘ゲーム、餓狼伝説シリーズ7作目、リアルバウトシリーズ3作目にあたる「リアルバウト餓狼伝説2 NEW COMMERS」のトレーニングモードを実現するためのLuaスクリプトです。
このプロジェクトは *table.save-0.94.lua* と *mame-rr-scripts* のカスタム版を含みます。


## スタートガイド

プロジェクトをダウンロードしてトレーニングモードを実行するまでの手順を説明します。


## 必要条件

トレーニングモードを実行するためには、以下のものが必要になります。

 1.  Luaスクリプトを実行できるアーケードゲームエミュレーター
	FBA-rr v0.0.7で開発と動作確認を行っています。
	次のリンクからダウンロードすることができます: http://tasvideos.org/EmulatorResources/Fbarr.html

2.  リアルバウト餓狼伝説2ロムデータ
	あなたの所持しているネオジオソフト「リアルバウト餓狼伝説2」（MVSまたはAES）からダンプしたデータを用意してください。
	私はソフトを所持していますがダンプ手段をもっていませんでした。
	しかし、私と同じ境遇の人でも https://www.humblebundle.com/ で販売されている、移植ソフトからデータを取り出すことができます。
	「NEOGEO CLASSICS PACK 1」を購入して、あなたの環境にインストールしたあと、インストールディレクトリのなかにロムデータを確認できるでしょう。
	BIOSデータも同じ場所から取り出すことができます。

3. ネオジオのBIOSデータ
	MVSかAESのネオジオ本体のBIOSチップからダンプするか、前述のロムデータと同じくhttps://www.humblebundle.com/ で販売されている、移植ソフトからデータを入手できます。
	BIOSのファイルを別名でコピーしなおす必要があります。
	FBA-rrではワーニングメッセージが表示されるものの、ゲームを起動できるようになります。

4. GDライブラリ
	このプロジェクトを作成するきっかけとなり、このプロジェクトにカスタマイズして含めているmame-rr-scriptsのScrollingInputDisplayでGDライブラリが必要になります。
	より詳しい情報はプロジェクトサイトを確認してください: http://luaforge.net/projects/lua-gd/
	GDライブラリは次のリンクからコピーしてください: http://luaforge.net/frs/download.php/1594/lua-gd-2.0.33r2-win32.zip

## 謝辞

ネオジオ オンラインコレクション 餓狼伝説バトルアーカイブズ2 のトレーニングモードに着想を得ました。
またその他の格闘ゲームのトレーニングモードを使用した経験からのアイデアも含んでいます。

レコーディング、リプレイ機能に関しては、ストリートファイター3rdストライクプレイヤーであるクロダ氏のトレーニングモードの動画からの影響もうけています。 
クロダ氏の動画は https://www.twitch.tv/kuroda236 で視聴することができます。
Youtubeでもクロダ氏の名前で検索すれば多くの動画を見つけることができでしょう。
迅速なレコーディング、リプレイを実現することが残っている課題の一つです。

mame-rr-scripts のすばらしいスクリプトの数々がなければ、このプロジェクトは存在しませんでした。
特にヒットボックスの表示と、OSDの表示については、プログラムコード解析の技術に劣る私にはできない仕事です。
素晴らしい仕事をありがとうございます。

シンプルで高機能なtable.saveスクリプトの存在も助かりました。
XML、JSONへのデータ保存も試してみましたが、table.saveの機能がこのプロジェクトには最適でした。

MAMEプロジェクト、FBAプロジェクト、各種チート、IPSパッチの開発者に感謝いたします。

リアルバウト餓狼伝説2の攻略サイトの情報がトレーニングのモチベーションを維持するのに役立ちました。
次のリンクから素晴らしい情報を見ることができます: http://gitr.web.fc2.com/

最後に、リアルバウト餓狼伝説2および餓狼伝説シリーズの開発者に感謝いたします。
